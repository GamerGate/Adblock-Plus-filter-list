GamerGate/Adblock-Plus-filter-list

This is the GamerGate Adblock Plus filter list.

It's an Adblock Plus compatible version of the GamerGate boycott list.

Once installed, it will automatically block all domains on the boycott list.

You can install it by clicking here. If clicking that link produces an error, try clicking here.

You can also view the filter list's contents, without installing it, by clicking here.

You'll need an Adblock Plus compatible extension installed on your browser before you can install the filter list. These extensions include Adblock Plus, Adblock Edge, and µBlock. Adblock Plus is available for Firefox, SeaMonkey, Chrome, Chromium, Opera, Safari, Internet Explorer, and Android. Adblock Edge is available for Firefox and SeaMonkey. µBlock is available for Chrome, Chromium, and Opera.
